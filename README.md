Praktikum 5 adalah penerapan berbagai widget UI bawaan Flutter untuk Material Design dan iOS Cupertino, yang secara bertahap memperkaya interaktivitas aplikasi. Mahasiswa memulai dengan membuat tombol menggunakan CupertinoButton dan CupertinoActivityIndicator, lalu beralih ke Material Design dengan mempraktikkan penggunaan FloatingActionButton (FAB) dan menyusun tata letak utama menggunakan Scaffold. Eksplorasi dilanjutkan dengan menambahkan interaksi pop-up melalui AlertDialog yang muncul saat tombol ditekan, serta mengimplementasikan form input dasar menggunakan TextField untuk menerima teks dari pengguna. Sebagai penutup, aplikasi diintegrasikan dengan showDatePicker, memungkinkan pengguna untuk memilih dan menampilkan tanggal tertentu.
